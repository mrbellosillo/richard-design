COLOR THEORY
Split Complementary Login UI
OBJECTIVE
Implement color harmony, contrast, and typography hierarchy in a login UI using split complementary color theory.
OVERVIEW

Designed a login screen using a split complementary color scheme as assigned by the design lead. The task required implementing color harmony, contrast, and typography hierarchy within a single UI component. After initial feedback on color role clarity and input field sizing, the design was revised and achieved a perfect final score. The modern approach — background blur, transparency, and a structured logo — became the defining visual choice of the composition.

KEY HIGHLIGHTS
  • Applied split complementary scheme: Orange as base, Yellow as supporting, Blue as accent
  • Used blue for both background and primary button to clearly separate it from the accent roles
  • Added an effect with background blur and transparency for visual depth
  • Established clear typography hierarchy from H1 title through labels, inputs, and CTA link
FINAL DESIGNS
DESIGN LEAD FEEDBACK

Good transparency and a blurry effect in the background. Effective branding and logo mockup with well-applied base color. The color scheme is applied perfectly.

REFLECTION

This task taught me that color role clarity — knowing which color is base, supporting, and accent — matters as much as the color selection itself. Getting feedback on role confusion before the revision made the second version significantly stronger.
