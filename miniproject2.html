TYPOGRAPHY
Typography-Only Homepage Hero
OBJECTIVE
Strengthen hierarchy, spacing, and visual emphasis using typography alone — no images or icons.
OVERVIEW

Designed a homepage hero section using only typography — no images, no icons, no decoration. The challenge was to establish hierarchy, emphasis, and visual balance using a single typeface at different weights and sizes. The design included an H1 headline, a supporting subtext, and a primary CTA button styled through text alone.

KEY HIGHLIGHTS
  • Established a clear H1 → subtext → CTA hierarchy using a single typeface with varied weights
  • Applied consistent leading and spacing to maintain visual rhythm across the composition
  • Used weight variation (Regular, Bold) to create emphasis
  • Achieved balance and alignment with correct text casing and spacing
FINAL DESIGNS
DESIGN LEAD FEEDBACK

No further revisions required. All instructions executed successfully. Demonstrated adherence to best practices in text placement, line spacing, and typographical sizing.

REFLECTION

Stripping away visual decoration forces you to rely on fundamentals. This task reminded me that good typography is already a complete design — scale, weight, and spacing can carry an entire hero section.
