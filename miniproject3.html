ACCESSIBILITY
Color-Blind Friendly UI Redesign
OBJECTIVE
Redesign a UI screen to improve accessibility for users with color-vision deficiencies while maintaining brand integrity.
OVERVIEW

Selected a UI screen with poor color choices and confusing visual cues, then redesigned it for full accessibility. The focus was on users with color-vision deficiencies — avoiding red/green conflicts, adding symbolic cues like icons and text labels, and using an accessible analogous color palette. The design maintained brand integrity by keeping original red and gray while expanding with orange hues.

KEY HIGHLIGHTS
  • Replaced red/green conflicts with color-blind-safe palette combinations
  • Added icons, shapes, and text labels alongside color to never rely on color alone
  • Expanded brand palette using an analogous color scheme (original red + orange hues)
  • Maintained typographic wordmark accuracy while redesigning all other visual elements

DESIGN LEAD FEEDBACK

Commendation for effective use of brand colors — maintaining original red and gray with creative expansion using an analogous color scheme. The background photo is well chosen and strongly relates to the webpage's concept.

REFLECTION

Accessibility isn't about removing color — it's about ensuring information is never locked behind it. Designing for color blindness improved the entire UI for all users, not just those with deficiencies.
