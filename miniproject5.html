INFORMATION ARCHITECTURE
Information Architecture & Navigation
OBJECTIVE
Organize app content using IA principles and appropriate UI components (tabs, cards, accordions) for easy findability.
OVERVIEW

Organized the Settings section of InboxPro using IA principles — findability, user-centered design, and accessibility. The five settings categories (Account Profile, Security & Privacy, Notifications, General Preferences, About App) were structured with tabs, cards, and accordions. Multiple screens were created for each tab, and both Light and Dark Mode visual identities were established.

KEY HIGHLIGHTS
  • Structured 5 settings categories with logical grouping and clear navigation hierarchy
  • Applied tabs, breadcrumbs, and accordions as appropriate IA components
  • Designed multiple sub-screens for each settings tab with consistent spacing
  • Created distinct Light Mode (saturated) and Dark Mode (desaturated) visual identities
FINAL DESIGNS
LIGHT MODES
DARK MODES
DESIGN LEAD FEEDBACK

The differentiation between Dark Mode and Light Mode is excellent, establishing clear, contrasting visual identities for both themes. Use saturated colors for light mode and desaturated for dark mode.

REFLECTION

Organizing information is a design decision. How you label, group, and sequence settings directly determines how quickly users can find what they need — IA is UX before the visuals start.
